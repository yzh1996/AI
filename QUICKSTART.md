# 快速开始指南

## 🚀 5 分钟上手

### 1️⃣ 安装依赖

```bash
pip install pymysql flask
```

### 2️⃣ 启动 Web 服务

```bash
python web_view_analyzer.py
```

看到以下输出表示启动成功：

```
============================================================
StarRocks 视图依赖关系 Web 分析工具
============================================================
数据库: donggua
访问地址: http://localhost:5000
============================================================
```

### 3️⃣ 打开浏览器

访问：http://localhost:5000

### 4️⃣ 开始分析

1. 在输入框输入视图名称，例如：`live_base_view`
2. 点击 "🔍 分析" 按钮
3. 查看依赖关系树
4. 点击视图节点（📊 图标）展开查看更深层的依赖

## 💡 使用技巧

### 图标说明
- 📊 = 视图（可点击展开）
- 📋 = 表（最底层，不可展开）
- ▶ = 可以展开
- ▼ = 已展开

### 颜色说明
- 🔵 蓝色标签 = 视图
- 🟢 绿色标签 = 表

### 操作说明
- **点击视图节点**：展开查看其依赖
- **再次点击**：折叠隐藏依赖
- **按 Enter 键**：快速触发分析

## ⚙️ 修改数据库配置

编辑 `web_view_analyzer.py` 文件，修改以下配置：

```python
DB_CONFIG = {
    'host': '你的数据库地址',
    'port': 2030,
    'database': '你的数据库名',
    'user': '你的用户名',
    'password': '你的密码'
}
```

## 🆘 常见问题

**Q: 提示"视图不存在"？**
A: 检查视图名称是否正确，确认在正确的数据库中

**Q: 无法连接数据库？**
A: 检查数据库地址、端口、用户名、密码是否正确

**Q: 页面加载很慢？**
A: 正常现象，复杂视图需要递归查询多次

**Q: 想用命令行版本？**
A: 运行 `python view_dependency_analyzer.py`
