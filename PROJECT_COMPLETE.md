## 🚀 立即开始使用

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动应用
```bash
python web_view_analyzer.py
```

### 3. 访问界面
打开浏览器：**http://localhost:5000**

### 4. 配置数据库
1. 点击右上角 **"⚙️ 数据库配置"**
2. 创建新配置并激活
3. 开始使用所有功能！
